Adding box plots

boxPlot :: (v ~ BaseSpace c, PointLike v n p, MonadState (Axis b c n) m, Plotable (BoxPlot v n) b, Foldable f, Enum n, TypeableFloat n) => f p -> State (Plot (BoxPlot v n) b) () -> m () Source

Add a BoxPlot to the AxisState from a data set.

  myaxis = r2Axis &~
    boxPlot data1

Example

mydata1 = [(1,3), (2,5.5), (3.2, 6), (3.5, 6.1)]
mydata2 = mydata1 & each . _1 *~ 0.5
mydata3 = [V2 1.2 2.7, V2 2 5.1, V2 3.2 2.6, V2 3.5 5]

myaxis :: Axis B V2 Double
myaxis = r2Axis &~ do
   boxPlot mydata1
   boxPlot mydata2
   boxPlot mydata3

Make a BoxPlot and take a State on the plot to alter its options

  myaxis = r2Axis &~ do
    boxPlot' pointData1 $ do
      fillBox .= True
      addLegendEntry "data 1"
